# 小小知识点
## 阻断函数
```cpp
getchar()	//等待键盘输入回车
sysem("pause")//任意键继续
```

## 常用函数
```cpp
printf("Hello World!")	//输出函数
atoi(arg)	//字符串转整数
Sleep(arg)	//等待

```
## 循环
```cpp
break;	//退出循环
continue;	//跳到循环首部,循环继续
```
## 其它
&arg 引用  
1.必须传变量  
2.会影响被引用的变量原本的值  

*arg 指针  
1.会影响被引用的变量原本的值  

abc:	冒号代表标签,常用于`goto`跳转
